== VidZen

The Ultimate Independent Movies, Web Series, Videos sharing website. The publishers not only get paid from advertisements but also through donations and payments from their viewer base. Dedicated Movie, Web Series, pages, and social networking.

To Set Up:

* Requirement for development: MySql database

* Run <tt>rake db:create</tt>

* Run <tt>rake db:migrate</tt>

* Add a config/database.yml file (database_skeleton.yml can be used as a skeleton)
